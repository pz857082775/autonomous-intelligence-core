# 模块 10：自动协调（多任务/多模块协同）

## 能力概述
多任务同时运行时自动分配资源，避免卡顿、冲突。多技能调用时自动协调顺序，跨软件操作时自动切换窗口/界面。

---

## 资源分配

### 资源类型
| 资源 | 管理方式 | 说明 |
|------|----------|------|
| CPU | 使用率限制 | 单任务不超过 80% |
| 内存 | 配额分配 | 按优先级分配 |
| 网络 | 带宽限制 | 避免占用全部带宽 |
| 磁盘 IO | 队列管理 | 避免 IO 冲突 |
| 工具实例 | 实例池 | 复用已打开的浏览器/会话 |

### 分配策略
```
任务到达
  ↓
评估资源需求
  ↓
检查可用资源
  ↓
分配资源配额
  ↓
监控资源使用
  ↓
动态调整配额
```

### 优先级队列
| 优先级 | 资源配额 | 示例任务 |
|--------|----------|----------|
| P0（紧急） | 不限制 | 系统修复、数据备份 |
| P1（高） | 60% | 用户指令、关键任务 |
| P2（中） | 30% | 自主计划任务 |
| P3（低） | 10% | 后台学习、整理 |

## 任务协调

### 并行任务管理
```
任务 A ─┬
       ├→ 资源协调器 → 分配资源 → 监控执行
任务 B ─┤
       └→ 冲突检测 → 解决冲突
任务 C ─┘
```

### 冲突检测
| 冲突类型 | 检测方式 | 解决策略 |
|----------|----------|----------|
| 资源冲突 | 资源需求超过可用 | 排队等待或降级 |
| 文件锁冲突 | 文件被占用 | 等待释放或报错 |
| 状态冲突 | 修改同一状态 | 串行执行 |
| 输出冲突 | 写入同一位置 | 合并或分开输出 |

### 协调规则
1. **互斥资源**：同一时间只允许一个任务使用
2. **共享资源**：按比例分配，设置上限
3. **依赖任务**：先执行前置任务
4. **独立任务**：可并行执行

## 技能调用协调

### 调用顺序协调
```
技能 A 需要输出给技能 B
  ↓
协调器确保 A 先执行
  ↓
A 完成后触发 B
  ↓
B 使用 A 的输出
```

### 技能实例管理
```
技能实例池：
- pdf: 2 个实例
- browser: 3 个实例
- exec: 5 个实例

任务请求实例
  ↓
池中有空闲 → 分配
  ↓
池中无空闲 → 等待或创建新实例
```

## 跨软件协调

### 窗口切换
```
需要操作软件 B
  ↓
检查软件 B 是否运行
  ↓
未运行 → 启动软件 B
  ↓
切换到软件 B 窗口
  ↓
执行操作
  ↓
返回原窗口（如需要）
```

### 界面无缝衔接
| 场景 | 协调方式 |
|------|----------|
| 浏览器→本地文件 | 下载后自动定位文件 |
| 本地文件→邮件 | 附件自动添加 |
| 多软件数据传递 | 剪贴板/临时文件中转 |

## 执行流协调

### 串行执行流
```
任务 1 → 任务 2 → 任务 3
  ↓      ↓      ↓
完成    完成    完成
```

### 并行执行流
```
任务 A ─┬
       ├→ 汇合 → 后续任务
任务 B ─┘
```

### 混合执行流
```
任务 1 → 任务 2 ─┬
                ├→ 任务 4 → 任务 5
任务 3 ─────────┘
```

## 稳定性保障

### 隔离机制
| 隔离类型 | 方式 | 说明 |
|----------|------|------|
| 进程隔离 | 独立进程 | 一个崩溃不影响其他 |
| 会话隔离 | 独立会话 | 状态不互相干扰 |
| 资源隔离 | 配额限制 | 一个不独占资源 |

### 降级策略
```
系统负载过高
  ↓
暂停低优先级任务
  ↓
释放资源给高优先级
  ↓
负载恢复正常后继续
```

### 恢复机制
```
任务失败
  ↓
释放占用资源
  ↓
记录失败状态
  ↓
通知协调器
  ↓
协调器重新调度
```

## 协调脚本

使用 `scripts/coordinator.py` 管理协调：

```bash
# 查看资源使用
python scripts/coordinator.py resources

# 查看任务队列
python scripts/coordinator.py queue

# 调整优先级
python scripts/coordinator.py prioritize --task task-id --priority P1

# 暂停任务
python scripts/coordinator.py pause --task task-id

# 恢复任务
python scripts/coordinator.py resume --task task-id
```
