# 模块 08：自动记忆（知识/经验存储）

## 能力概述
长期记忆存储任务历史、操作规则、软件特征、界面布局、用户偏好；短期记忆记录当前任务上下文。记忆自动索引、自动更新。

---

## 记忆分类

### 长期记忆
| 记忆类型 | 内容 | 存储位置 |
|----------|------|----------|
| 任务历史 | 已完成任务、结果、耗时 | `memory/archive/` |
| 操作规则 | 操作流程、最佳实践 | `MEMORY.md` |
| 软件特征 | 软件用法、配置、限制 | `references/software/` |
| 界面布局 | 常用界面元素位置 | `references/ui-layouts/` |
| 用户偏好 | 用户习惯、偏好设置 | `MEMORY.md` |
| 知识图谱 | 实体关系、依赖 | `ontology/` |

### 短期记忆
| 记忆类型 | 内容 | 存储位置 |
|----------|------|----------|
| 任务上下文 | 当前任务目标、进度 | `temp/current-task.md` |
| 执行状态 | 当前步骤、变量、中间结果 | `temp/state.json` |
| 临时数据 | 临时文件、缓存 | `temp/cache/` |

---

## 记忆存储结构

### 目录结构
```
~/.openclaw/workspace/
├── MEMORY.md                    # 核心长期记忆
├── memory/
│   ├── YYYY-MM-DD.md           # 每日记忆日志
│   └── archive/                # 归档记忆
├── temp/
│   ├── current-task.md         # 当前任务
│   ├── state.json              # 执行状态
│   └── cache/                  # 临时缓存
└── references/
    ├── software/               # 软件知识
    ├── ui-layouts/             # 界面布局
    └── workflows/              # 工作流程
```

### 记忆条目格式
```markdown
## [记忆标题]

**类型**：[任务历史/操作规则/软件特征/界面布局/用户偏好]
**创建时间**：YYYY-MM-DD HH:MM
**最后更新**：YYYY-MM-DD HH:MM
**标签**：[标签 1, 标签 2, ...]
**内容**：
[记忆内容]

**关联**：
- 相关记忆：[链接]
- 相关任务：[链接]
- 相关技能：[链接]
```

## 记忆写入

### 自动写入触发
| 触发事件 | 写入内容 |
|----------|----------|
| 任务完成 | 任务结果、经验教训 |
| 学习完成 | 新知识、新技能 |
| 修复完成 | 修复记录、避免方法 |
| 优化完成 | 优化内容、效果对比 |
| 用户交互 | 用户偏好、反馈 |

### 写入流程
```
事件发生
  ↓
格式化记忆内容
  ↓
选择存储位置（长期/短期）
  ↓
写入文件
  ↓
更新索引
  ↓
验证写入成功
```

## 记忆检索

### 检索方式
| 检索方式 | 工具 | 说明 |
|----------|------|------|
| 关键词搜索 | `memory_search` | 语义搜索 |
| 路径读取 | `memory_get` | 精确读取 |
| 标签过滤 | 自定义脚本 | 按标签筛选 |
| 时间范围 | 自定义脚本 | 按时间筛选 |

### 检索脚本
```bash
# 语义搜索
python scripts/memory_manager.py search --query "关键词" --limit 10

# 按标签检索
python scripts/memory_manager.py query --tags "标签 1,标签 2"

# 按时间检索
python scripts/memory_manager.py timeline --from 2026-01-01 --to 2026-03-14
```

### 毫秒级检索
- 使用倒排索引加速
- 缓存热点记忆
- 预加载常用记忆

## 记忆更新

### 更新策略
| 更新类型 | 策略 | 说明 |
|----------|------|------|
| 追加 | 添加新内容 | 不修改原有内容 |
| 覆盖 | 替换旧内容 | 内容已过期/错误 |
| 归档 | 移动到老文件 | 内容不再常用 |
| 删除 | 移除内容 | 内容无效/敏感 |

### 自动归档
```
定期检查记忆文件（每周）
  ↓
识别低频访问记忆（30 天未访问）
  ↓
移动到 archive 目录
  ↓
更新索引
```

### 记忆过期
| 记忆类型 | 有效期 | 过期处理 |
|----------|--------|----------|
| 临时数据 | 1 天 | 自动删除 |
| 任务历史 | 90 天 | 归档 |
| 操作规则 | 永久 | 更新时覆盖 |
| 用户偏好 | 永久 | 持续更新 |

## 记忆索引

### 索引结构
```json
{
  "entries": [
    {
      "id": "mem-001",
      "path": "MEMORY.md",
      "line": 10,
      "type": "操作规则",
      "tags": ["pdf", "编辑"],
      "created": "2026-03-14",
      "accessed": "2026-03-14",
      "keywords": ["pdf", "旋转", "编辑"]
    }
  ],
  "tags": {
    "pdf": ["mem-001", "mem-002"],
    "编辑": ["mem-001", "mem-003"]
  }
}
```

### 索引维护
- 写入时自动更新索引
- 定期重建索引（每天）
- 索引缓存到内存

## 记忆脚本

使用 `scripts/memory_manager.py` 管理记忆：

```bash
# 写入记忆
python scripts/memory_manager.py write --type long-term --content "记忆内容" --tags "标签"

# 搜索记忆
python scripts/memory_manager.py search --query "关键词" --limit 10

# 整理记忆
python scripts/memory_manager.py compact --archive-older-than 30

# 导出记忆
python scripts/memory_manager.py export --output memory-backup.json
```
